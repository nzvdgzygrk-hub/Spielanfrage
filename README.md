# Spielanfrage

Kleine iPhone-taugliche App für Spielanfragen der SSVg Velbert.

## Funktionen

- Standardtext ist bereits eingetragen und kann jederzeit geändert werden.
- Zeitraum auswählen und Samstage, Sonntage oder beide Tage laden.
- Pro Tag auswählen: offen, Heim, Auswärts oder streichen.
- WhatsApp-Text direkt kopieren oder teilen.
- Eingaben werden im Browser gespeichert.
